# 🎬 MovieTracker

MovieTracker, kullanıcıların film araması yapabildiği, detaylarını görebildiği ve favorilere ekleyebildiği bir mobil uygulamadır.

## 🚀 Kullanılan Teknolojiler
- Kotlin
- Coroutines
- Retrofit
- TMDB API
- Room Database
- MVVM Architecture
